# Bangkit Capstone - Machine Learning Service Team C23-PS072

## How to Install

### Requirements
- node.js 18.x
- python 3.9.x
- pip 20.x
- librosa (python dependencies)
- matplotlib (python dependencies)
- numpy (python dependencies)
- npm / yarn

### Steps to run
- Install node dependencies
```bash
npm install
```

- Run service
```bash
node index.js
```
